📺 Changelog Popup — Show for Everyone
Full-screen What's New popup on next visit

Choose what to show users when they next visit the site. This clears their "seen" flag so the popup appears again.

  1. Go to Firebase Console → your project
  2. Click Messaging → New Campaign
  3. Choose Firebase Notification messages
  4. Set title, body, optionally a click URL
  5. Target → select your web app → Send

Users who clicked Allow on the notification prompt receive push notifs. iOS requires adding site to home screen first.

How to Enable Maintenance Mode

1. Go to your Vercel Dashboard

2. Select your CineRealm project → Settings → Environment Variables

3. Add: MAINTENANCE_MODE = true

4. Redeploy — users will see maintenance page

5. Set back to false and redeploy to go live again
